# Vue + ElementUI 后台管理系统框架

## **项目使用**
``` bash
# 安装项目依赖
npm install

# 开启本地服务，默认为localhost:9000
npm run dev

# 项目打包，构建生产环境
npm run build

# 打包过程中想查看具体报告则可以通过以下命令实现
npm run build --report
```

## **组件文档**
``` bash
# 全局组件路径
src/components/GlobalComponents

# 功能说明

# 按钮组件
VButton
# 表格组件
VTable
# 远程实时搜索模糊查询下拉数据组件
VAutoComplete
# 数字及小数输入框组件
VNumber
